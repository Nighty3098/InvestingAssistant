> ONLY FOR PYTHON 3.12

#### Install Poetry:


```
curl -sSL https://install.python-poetry.org | python3 -
```

#### Installing


```
git clone https://github.com/Nighty3098/InvestingAssistant
cd InvestingAssistant
poetry shell
poetry install
cd src
```

#### In .env file


```
API_ID = ''
API_HASH = ''
BOT_TOKEN = ""
DEVELOPER_USERNAME = ""
```

#### Run


```
python3 main.py
```
