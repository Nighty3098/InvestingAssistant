#### Install Poetry:


```
curl -sSL https://install.python-poetry.org | python3 -
```

#### Installing


```
git clone https://github.com/Nighty3098/InvestingAssistant
cd InvestingAssistant
poetry shell
poetry install
cd src
```

#### In .env file


```
API_ID = ''
API_HASH = ''
BOT_TOKEN = ""
```

#### Run


```
python3 main.py
```
